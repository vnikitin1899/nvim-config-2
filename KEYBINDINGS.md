# Комбинации клавиш Neovim

## Leader клавиша
- `<leader>` = `Пробел` (Space)

---

## Поиск и навигация по файлам

### Telescope
- `ff` - Поиск файлов (Telescope)
- `fg` - Поиск текста в файлах (без node_modules)
- `<leader>fg` - Поиск текста в файлах (включая node_modules)
- `ag` - Поиск текста в файлах (включая node_modules, альтернатива)

---

## LSP (Language Server Protocol)

**Работают только в буферах с активным LSP клиентом (например, TypeScript файлы)**

### Навигация
- `gd` - Перейти к определению (моментальный переход к первому результату)
- `gD` - Перейти к объявлению
- `gi` - Перейти к реализации
- `K` - Показать документацию (hover)

### Диагностика
- `[d` - Перейти к предыдущей ошибке/предупреждению (с показом деталей)
- `]d` - Перейти к следующей ошибке/предупреждению (с показом деталей)
- `gl` - Показать детали ошибки в floating window

### Рефакторинг
- `<leader>rn` - Переименовать символ
- `<leader>ca` - Code action (быстрые исправления)

### Справка
- `<C-k>` - Показать сигнатуру функции (в Insert режиме)

---

## Treesitter (инкрементальное выделение)

- `gnn` - Начать выделение узла
- `grn` - Расширить выделение до следующего узла
- `grc` - Расширить выделение до области видимости
- `grm` - Уменьшить выделение

---

## Git

### Git Diff (кастомная панель)
- `<leader>gd` - Открыть панель git diff (список файлов слева, diff справа)

**В панели git diff:**
- `<CR>` - Выбрать файл из списка
- `j` - Перейти к следующему файлу
- `k` - Перейти к предыдущему файлу
- `q` - Закрыть панель

**В diff буфере:**
- `j` - Перейти к следующему файлу
- `k` - Перейти к предыдущему файлу
- `q` - Закрыть панель

### Diffview
- `<leader>gv` - Открыть Diffview со всеми изменениями
- `<leader>gh` - Открыть историю файла (Diffview)
- `<leader>gq` - Закрыть Diffview

---

## Автоматическое сохранение

- Автоматическое сохранение при выходе из Insert режима (`InsertLeave`)

---

## Примечания

- Все LSP маппинги работают только в буферах с активным LSP клиентом
- `<leader>` = пробел (Space)
- Маппинги для git diff панели работают только внутри этой панели
- Treesitter маппинги работают для всех языков с поддержкой treesitter
