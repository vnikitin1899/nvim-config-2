# Памятка по горячим клавишам

- `<leader>` = пробел.
- `:KeymapHelp` или `<leader>hk` мгновенно открывают этот файл (используется Glow, если плагин установлен).

---

## Базовая навигация (Normal mode)
- `h / j / k / l` — перемещение по символам и строкам.
- `w / b / e` — вперёд/назад по словам, к концу слова.
- `0 / ^ / $` — в начало строки, к первому непустому символу, в конец строки.
- `gg / G` — в начало/конец файла.
- `<C-d> / <C-u>` — полстраницы вниз/вверх.
- `<C-f> / <C-b>` — целая страница вниз/вверх.
- `zz / zt / zb` — текущая строка по центру / сверху / снизу.
- `<C-o> / <C-i>` — назад/вперёд по истории прыжков.
- `/pattern` и `?pattern` — поиск вперёд/назад, `n / N` — следующий/предыдущий результат.
- `* / #` — найти следующее/предыдущее вхождение слова под курсором.

---

## Окна и буферы
- `wh / wl / wj / wk` — перейти в окно слева/справа/снизу/сверху (`<C-w>` + направление под капотом).
- `<leader>e` — переключить Neo-tree файлового менеджера (слева, от cwd).
- `<leader>ef` — подсветить текущий файл в дереве.
- `<leader>ec` — закрыть дерево.

**Размеры окон**
- `<C-w>> / <C-w><` — расширить/сузить текущий сплит по горизонтали.
- `<C-w>+ / <C-w>-` — увеличить/уменьшить высоту сплита.
- `:resize ±N` — задать высоту текущего окна (например, `:resize +5`).
- `:vertical resize ±N` — аналог для ширины (`:vert resize -8` и т.д.).

---

## Поиск и работа с проектом
**Telescope**
- `ff` — поиск файлов.
- `fg` — live grep без `node_modules`.
- `<leader>fg` — live grep по всему проекту, включая скрытые и `node_modules`.
- `ag` — альтернативный live grep по всему проекту.

**Treesitter Context**
- `<leader>uc` — включить/выключить фиксацию текущего контекста.

**Глобальный поиск и замена (Spectre)**
- `ss` — открыть Spectre для проекта.
- `<leader>sw` (normal) — искать слово под курсором.
- `<leader>sw` (visual) — искать/заменять выделение.

**Silicon (скриншоты кода)**
- `<leader>ss` (visual) — сделать снимок выделения.
- `<leader>sb` — снять текущий буфер в буфер обмена.

## Форматирование
- `<leader>cf` — форматировать текущий буфер/выделение через Conform.
- `:FormatDisable[!]` / `:FormatEnable[!]` / `:FormatToggle[!]` — управление автоформатированием (без `!` — глобально, с `!` — только для буфера).
- Prettier автоматически запускается при каждом выходе из Insert для JS/TS/Markdown/CSS/HTML буферов (JSON только вручную через `<leader>cf` / `:Format`).
- `:FormatInsertDisable[!]` / `:FormatInsertEnable[!]` / `:FormatInsertToggle[!]` — управление автозапуском Prettier на `InsertLeave`.

## Markdown
- `<leader>mg` — предпросмотр текущего markdown-файла через Glow.

## Git и Diff
**LazyGit**
- `<leader>gd` — открыть LazyGit в плавающем окне.
- `git` — быстрый вызов LazyGit (без `<leader>`).

**Diffview**
- `<leader>gv` — открыть общий diff.
- `<leader>gh` — история текущего файла.
- `<leader>gq` — закрыть Diffview.

**Git Conflict**
- `]x / [x` — перейти к следующему/предыдущему конфликту.
- `co / ci / cb` — принять текущую, входящую или обе части.
- `c0` — откатить выбор для текущего конфликта.
- `<leader>gx` — обновить подсветку конфликтов после фиксов.

---

## LSP (в буферах с активным сервером)
**Навигация и помощь**
- `gd` — перейти к определению (сразу к первому результату).
- `gD` — перейти к объявлению.
- `gi` — перейти к реализации.
- `K` — hover-документация.
- `<C-k>` (insert/normal) — подсказка сигнатуры.

**Диагностика**
- `[d / ]d` — предыдущая/следующая проблема с автоматическим всплывающим окном.
- `gl` — показать подробности ошибки у курсора.

**Рефакторинг**
- `<leader>rn` — переименовать символ.
- `<leader>ca` — применить доступные code actions.
- `<leader>fx` — автоматически применить первую доступную правку/quickfix (включая автоимпорт).
- `:LspAutoImport` — добавить или упорядочить импорты через LSP.

---

## Полезные заметки
- Автосохранение срабатывает при выходе из Insert (`InsertLeave`).
- Маппинги git-панели (`o/t/u`) активны только пока открыто кастомное окно diff.
- Treesitter-переключатель (`<leader>uc`) работает для всех языков, где включён treesitter.
- Если Glow недоступен, `:KeymapHelp` просто откроет файл `KEYBINDINGS.md` в новой вкладке.
